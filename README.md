Doris Kuo | Portfolio Website

朵莉絲的個人作品集網站

✨ Overview | 概要

EN: A one-page interactive portfolio built with HTML, CSS (Sass), JavaScript, and jQuery. It features parallax scrolling via Skrollr and a swipeable skills section via Swiper. The site supports English/Chinese toggle, and includes subpages for resume, learning journey, and career story.
中文： 使用 HTML、CSS（Sass）、JavaScript、jQuery 製作的一頁式互動作品集。整站透過 Skrollr 實現視差滾動，並用 Swiper 建立可滑動的技能展示。支援 中英文切換，並提供履歷、學習歷程與轉職故事等子頁。

🚀 Features | 功能特色

EN: One-page layout with smooth anchor navigation

EN: Parallax & scroll interactions powered by Skrollr
中文： 使用 Skrollr 實現視差與滾動互動

EN: Skills carousel implemented with Swiper (slider)
中文： 技能區塊使用 Swiper 製作輪播

EN: Multilingual toggle (EN / ZH)
中文： 提供中英文切換

EN: Modular subpages (resume, catalyst, learning, change career)

EN: Responsive layout, hover animations, icon set via Font Awesome
