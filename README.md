👻 Micro Horror Game (Vue.js)

微恐怖遊戲（Vue.js）

🕹️ Introduction | 介紹

EN:
A lightweight web-based horror mini-game built with Vue.js.
This project demonstrates interactive UI design, event handling, and animation effects, showcasing how front-end frameworks can create immersive game experiences.

中文：
一個以 Vue.js 開發的輕量級網頁恐怖小遊戲。
此專案展示了互動式 UI 設計、事件處理與動畫效果，呈現前端框架如何打造沉浸式遊戲體驗。

⚙️ Features | 功能特色

EN: Vue.js component-based architecture for maintainability

中文： 採用 Vue.js 元件化架構，方便維護與擴充

EN: Interactive event handling (click, hover, keyboard inputs)

中文： 支援互動事件（點擊、滑鼠移入、鍵盤操作）

EN: Smooth animation effects and horror-themed UI design

中文： 流暢的動畫效果與恐怖風格介面設計
